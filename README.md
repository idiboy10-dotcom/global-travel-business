# 🌍 Global Travel Business

Bienvenue sur le dépôt officiel de **Global Travel Business**, une agence de voyages internationale basée à Dakar, Sénégal.

## ✨ À propos
Global Travel Business propose des séjours premium en Afrique et en Europe :
- Séjour culturel à l’Île de Gorée (Sénégal)
- Safari au Kenya
- Voyage romantique à Paris

## 🚀 Déploiement
Ce site est développé avec **Streamlit** et déployé sur **Streamlit Cloud**.

### Installation locale
Clonez le dépôt et installez les dépendances :
```bash
git clone https://github.com/idiboy10-dotcom/global-travel-business.git
cd global-travel-business
pip install -r requirements.txt
streamlit run mon_site.py
